Project Overview
PhotoVista is a responsive, single-page website created to demonstrate modern
web layout techniques using Flexbox and CSS Grid.  
The project focuses on clean UI design, responsiveness, and structured layouts
commonly used in real-world web applications.

Objectives Achieved
- Built a responsive webpage using HTML5 and CSS3
- Implemented layouts using Flexbox (navigation, hero, features, contact)
- Implemented layouts using CSS Grid (gallery and blog sections)
- Designed a multi-section webpage with smooth navigation
- Ensured compatibility across different screen sizes

Features
- Responsive navigation bar with mobile menu
- Hero section with call-to-action
- Photo gallery using CSS Grid
- Blog section with featured and regular articles
- About / features section
- Contact section with newsletter subscription UI
- Footer with social links
- Smooth scrolling and mobile menu interaction using JavaScript

Technologies Used
- HTML5
- CSS3 (Flexbox & CSS Grid)
- JavaScript (basic interactivity)
- Google Fonts
- Font Awesome Icons

 How to Run the Project
1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Resize the browser window to see the responsive behavior.
